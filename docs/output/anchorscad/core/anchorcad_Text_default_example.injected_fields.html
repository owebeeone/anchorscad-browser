Datatree Injected Fields for Text

Generates 3D text.
Args:
text: Text string to render.
size: Height of text
depth: Depth of text
font: Font name for rendering.
halign: Horizontal alignment, left, center, centre or right
valign: Vertical alignment. top, center, centre, baseline or bottom
spacing: Character spacing.
direction: Direction of rendering, ltr, rtl, ttb or btt
language: Language being rendered.
script: Script being rendered.
fn: fixed number of segments. Overrides fa and fs
fs: minimum length of each segment
fa: minimum angle (in degrees) of each segment
Other args:
offset_r, offset_delta, offset_chamfer, offset__fa, offset__fs, offset__fn
