Datatree Injected Fields for EdgeBracket

Bracket for holding a flat object at the edge. This specifically was designed
to replace the plastic brackets that hold a mirror in place on a make-up table
which had become brittle and needed replacement.

Args:
w: Width of edge bracket
l: Overall length of edge bracket
tw: Tab width
sw: separator width
fn: The number of facets to use for the extrusion.
fa: minimum angle (in degrees) of each segment
fs: minimum length of each segment
Other args:
screw_hole_outer_delta, screw_hole_counter_sink_overlap, screw_hole_dia, screw_hole_access_len, screw_hole_access_dia, screw_hole_outer_dia, screw_hole_dims, screw_hole_overlap_delta
