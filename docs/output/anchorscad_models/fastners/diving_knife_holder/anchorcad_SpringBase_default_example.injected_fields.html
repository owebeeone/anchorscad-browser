Datatree Injected Fields for SpringBase

The main shape of the handle attachment with adjustable number of spring loops.
Args:
scaffold_r: Radius of the stadium
scaffold_inner_r: Offset of the stadium path
scaffold_w: Width of the stadium
scaffold_t: Top width of the stadium
scaffold_offset: Offset of the stadium path
scaffold_square_left: Square the left side of the stadium
scaffold_square_right: Square the right side of the stadium
scaffold_h: The height of the extrusion.
scaffold_lin_extrude_twist: The twist of the extrusion in degrees.
scaffold_lin_extrude_slices: The number of slices to use for the extrusion if twist is applied.
scaffold_lin_extrude_scale: The scale of the extrusion in X and Y.
fn: None
scaffold_lin_extrude_path_fn: The number of facets to use for the extrusion.
scaffold_lin_extrude_use_polyhedrons: If true will use polyhedrons to generate the extrusion.
scaffold_rot_extrude_convexity: Openscad convexity parameter.
scaffold_rot_extrude_use_polyhedrons: Use polyhedrons instead of rotate_extrude.
