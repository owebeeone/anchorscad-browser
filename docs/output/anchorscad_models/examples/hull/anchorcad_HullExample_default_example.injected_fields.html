Datatree Injected Fields for HullExample

An example using OpenSCAD hull.

Note that no anchors are provided for the surface resulting from
the generated hull surface.

Args:
r: Radius of sphere
fn: fixed number of segments. Overrides fa and fs
fa: minimum angle (in degrees) of each segment
fs: minimum length of each segment
Other args:
x, y, z
