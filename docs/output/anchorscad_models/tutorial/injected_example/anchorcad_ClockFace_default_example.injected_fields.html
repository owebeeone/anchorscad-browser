Datatree Injected Fields for ClockFace

Simple clock face with different sized spheres at the points.
Args:
count_points: The number of points on the clock face
r_mod_map: Radius at each point
x_pos: Radius of the clock face centres
r: Radius of sphere
fn: fixed number of segments. Overrides fa and fs
fa: minimum angle (in degrees) of each segment
fs: minimum length of each segment
