A 3x hinge bar with holes for the hinge.

Args:
epsilon: Fudge factor

nib_ang_len: Angle length of inner hinge nib

fa: minimum angle (in degrees) of each segment
fs: minimum length of each segment
rx_use_polyhedrons: Use polyhedrons instead of rotate_extrude.
bar_h: Height of hinge bar
joint_offset: Offset of hinge hole from end of bar
