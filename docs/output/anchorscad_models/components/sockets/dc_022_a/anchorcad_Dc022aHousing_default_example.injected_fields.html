Datatree Injected Fields for Dc022aHousing

A housing for a DC-022-2.5A-2.0 socket.
Args:
top_r: Top radius of the socket
top_h: Height of top of the socket
fn: fixed number of segments. Overrides fa and fs
fa: minimum angle (in degrees) of each segment
fs: minimum length of each segment
top_access_hole_r: Front access hole radius, only used for clearance for inserting the jack and plug.
top_access_hole_h: Height of rear access hole
body_r: Body radius of the socket
body_flat_r: Flat radius of the socket
body_h: Height of body of the socket
body_twist: The twist of the extrusion in degrees.
body_slices: The number of slices to use for the extrusion if twist is applied.
body_scale: The scale of the extrusion in X and Y.
body_path_fn: The number of facets to use for the extrusion.
body_use_polyhedrons: If true will use polyhedrons to generate the extrusion.
rear_access_hole_r: Rear access hole radius, must take mounting bolt
rear_access_hole_h: Height of rear access hole
plug_r: Plug radius of the socket
plug_h: Height of plug hole of the socket
add_access_holes: Add access holes for inserting the plug and jack
thickness: Thickness of the housing wall
shell_thickness: Thickness of the housing shell
extension: Extension of the housing beyond the socket
extender_t: Top width of the stadium
extender_offset: Offset of the stadium path
extender_square_left: Square the left side of the stadium
extender_lin_extrude_twist: The twist of the extrusion in degrees.
extender_lin_extrude_slices: The number of slices to use for the extrusion if twist is applied.
extender_lin_extrude_scale: The scale of the extrusion in X and Y.
extender_lin_extrude_path_fn: The number of facets to use for the extrusion.
extender_lin_extrude_use_polyhedrons: If true will use polyhedrons to generate the extrusion.
Other args:
