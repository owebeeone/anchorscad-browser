Datatree Injected Fields for ArduinoNanoClassic

Outline for the classic Arduino Nano board.
Pins are numbered from 1 to 30. Mount holes are numbered from 1 to 4
in the same order as pins.

Args:
pcb_size: The (x,y,z) size of PCB for the Arduino Nano Classic board
mount_r: Radius of cylinder.
fn: fixed number of segments. Overrides fa and fs
fa: minimum angle (in degrees) of each segment
fs: minimum length of each segment
small_fn: fixed number of segments. Overrides fa and fs
pin_hole_r: Radius of cylinder.
icsp_header_w: Length of prism sides. Overrides parameter r
icsp_header_taper_l: The height of the extrusion.
icsp_header_base_h: The height of the extrusion.
icsp_header_base_twist: The twist of the extrusion in degrees.
icsp_header_base_slices: The number of slices to use for the extrusion if twist is applied.
icsp_header_base_scale: The scale of the extrusion in X and Y.
icsp_header_base_path_fn: The number of facets to use for the extrusion.
icsp_header_base_use_polyhedrons: If true will use polyhedrons to generate the extrusion.
usb_size: None
Other args:
pitch, icsp_header_pins_x, icsp_header_pins_y, icsp_header_solder_side_l, icsp_header_l, icsp_header_base_w, icsp_header_base_c, usb_anchor, epsilon
