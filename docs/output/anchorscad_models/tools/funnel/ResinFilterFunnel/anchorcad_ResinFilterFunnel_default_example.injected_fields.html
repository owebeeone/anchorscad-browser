Datatree Injected Fields for ResinFilterFunnel

Funnel for disposable mesh + paper filters typically used for SLA resin
being returned to the resin bottle. This model fits the Anycubic bottle
and should snap fit onto the groove near the mouth of the bottle.

Args:
fn: fixed number of segments. Overrides fa and fs
fa: minimum angle (in degrees) of each segment
fs: minimum length of each segment
h_boss: Height of cone
fn_boss: fixed number of segments. Overrides fa and fs
r_base_boss: Base radius
r_top_boss: Top radius
r_base_inner_boss: Base radius
r_top_inner_boss: Top radius
fa_boss: minimum angle (in degrees) of each segment
fs_boss: minimum length of each segment
Other args:
h, r_inner_base, r_inner_top, l_side, t, h_adapter_mesh, r_top_outer_mesh, r_top_inner_mesh, h_adapter, r_top_outer_adapter, r_top_inner_adapter, h_bottle_inner, r_bottle_inner, t_base_bottle_inner, t_top_bottle_inner, h_bottle_outer, r_bottle_outer, t_bottle_tail, h_rim, t_rim, w_tab, d_tab, a_tab1, a_tab2, epsilon, r_boss, p_boss, n_boss, t_boss, t_base_boss, t_top_boss, epsilon_boss, show_cutaway
