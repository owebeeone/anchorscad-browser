Datatree Injected Fields for BatteryAdapter

Battery adapter for cylindrical batteries. This assumes the batteries
are the same length but different diameters.

Args:
count: Number of batteries
l: Length of battery
margin: Margin at the ends of the batteries
r_small: Radius of the smaller battery
r_large: Radius of the larger battery
compensation: Compensation for FDM printer
gap_angle: Gap of the enlosing cylinder in angle
twist: The twist of the extrusion in degrees.
slices: The number of slices to use for the extrusion if twist is applied.
scale: The scale of the extrusion in X and Y.
fn: Number of facets for the linear extrusion
path_fn: The number of facets to use for the extrusion.
use_polyhedrons: If true will use polyhedrons to generate the extrusion.
