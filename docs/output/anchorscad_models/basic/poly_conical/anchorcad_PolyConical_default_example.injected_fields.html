Datatree Injected Fields for PolyConical

A circular polycone shape. The segments are defined by
a tuple of Segment objects.

Args:
segments: Tuple of Segments defining shape
angle: The sweep angle to extrude.
convexity: Openscad convexity parameter.
use_polyhedrons: Use polyhedrons instead of rotate_extrude.
Other args:
path_fn, fn, fa, fs
