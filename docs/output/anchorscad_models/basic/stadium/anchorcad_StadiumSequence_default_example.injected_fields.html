Datatree Injected Fields for StadiumSequence

Assembles a sequence of stadium prisms and revolutions base-to-top.
Args:
r: Radius of the stadium
w: Width of the stadium
t: Top width of the stadium
offset: Offset of the stadium path
square_left: Square the left side of the stadium
square_right: Square the right side of the stadium
h: The height of the extrusion.
lin_extrude_twist: The twist of the extrusion in degrees.
lin_extrude_slices: The number of slices to use for the extrusion if twist is applied.
lin_extrude_scale: The scale of the extrusion in X and Y.
fn: The number of facets to use for the extrusion.
lin_extrude_path_fn: The number of facets to use for the extrusion.
lin_extrude_use_polyhedrons: If true will use polyhedrons to generate the extrusion.
inner_r: Offset of the stadium path
rot_extrude_convexity: Openscad convexity parameter.
rot_extrude_use_polyhedrons: Use polyhedrons instead of rotate_extrude.
Other args:
hide_cage, sweep_angle, rot_extrude_path_fn, fa, fs, sequence
